## Technology stack

-   Language: [TypeScript v4][2]
-   Component engine: [WebCell v2][3]
-   Component suite: [BootStrap v4][4]
-   Icon font: [FontAwesome v5][5]
-   State management: [MobX v5][6]
-   Package bundler: [Parcel v1][7]

## Local development

```shell
npm install
npm start
```
