# Momo Chat

## Technology stack

-   Language: [TypeScript v3][1]
-   Component engine: [WebCell v2][2]
-   Component suite: [BootStrap v4][3]
-   Icon font: [FontAwesome v5][4]
-   State management: [MobX v5][5]
-   Package bundler: [Parcel v1][6]

## Local development

```shell
npm install
npm start
```
